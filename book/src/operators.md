# Operators

Possible operators:

Match
```
=
```

Equality
```
=:=
==
```

Not equal:
```
=/=
/=
```

Less than/Less than or equal:
```
<
<=
```

Greater than/Greater than or equal:
```
>
>=
```

Addition:
```
+
```

Subtraction or negative:
```
-
```

Multiplication:
```
*
```

Division:
```
/
```

Concatenation:
```
++
```

Cons or Map Update:
```
|
```
